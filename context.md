# RouteFlo - Route Management for Mat Delivery

## Product Overview
RouteFlo is route management software built specifically for mat and linen delivery businesses. It helps operations managers stop using spreadsheets and phone calls to manage their drivers.

**Target Customer:** Mat rental companies, linen services, uniform delivery - businesses like Cintas competitors

## Key Features
1. **Delivery Stops Management** - Add customers, set service days (Mon/Wed/Fri etc), assign service packages, CSV upload
2. **Route Generation** - Pick a date, confirm stops, auto-generate optimized routes
3. **Map View** - Visual route display, color-coded by driver, edit stops from map
4. **Calendar View** - See driver schedules by day/week/month
5. **Materials Tracking** - Mat inventory, stock quantities, assignments
6. **Driver Mobile App** - Simple daily schedule with Google Maps integration

## Target Audience
- Operations managers at mat/linen delivery companies (5-50 drivers)
- Not technical people - speak plainly
- Currently using spreadsheets, paper, or "someone's head" for routes

## Demo Customer
Grizzly Mats (Baltimore area) - used for screenshots showing the app with real-looking data
