## DESIGN NOTES FOR PAGE BUILDERS:

### Color Scheme:
- **Primary Blue:** #1e3a8a (dark blue for headlines)
- **Accent Blue:** #3b82f6 (bright blue for buttons/highlights)
- **CTA Red:** #ef4444 (red for main CTA buttons)
- **Success Green:** #059669 (for time saved badges)
- **Text Gray:** #4b5563 (body copy)
- **Background:** #f8f9fa (light gray)

### Typography:
- **Headlines:** Bold, 2-2.5x body text size
- **Subheadlines:** 1.3x body text size
- **Body:** 16-18px, 1.6 line height for readability

### Layout Suggestions:
1. **Hero section** at top with gradient blue background
2. **Lead form** should stand out (white box with shadow, centered)
3. **Tool cards** in grid or stack (mobile)
4. **Testimonials** with light background boxes
5. **FAQ** accordion style (optional) or simple list
6. **Final CTA** with blue gradient background
7. **Form should appear above the fold** and repeat at bottom

### Mobile Optimization:
- Stack tool cards vertically
- Make form fields full width
- Reduce heading sizes for mobile
- Ensure CTA button is easily tappable (minimum 44px height)

---

## CONVERSION OPTIMIZATION TIPS:

1. **Above the fold:** Hero + Form visible immediately
2. **Clear value prop:** "5 tools that fix 5 problems" (specific)
3. **Social proof:** Testimonials from real people with results
4. **Objection handling:** FAQ answers "is this free?" "is it hard?" etc.
5. **Multiple CTAs:** Form at top, link to form in middle, form at bottom
6. **Low friction:** Only ask for First Name + Email (that's it)
7. **Trust signals:** Privacy note, real testimonials, no credit card required
